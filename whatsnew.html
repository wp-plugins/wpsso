WPSSO version 2.5 includes two new features -- a 'Social Settings' metabox for the user profile page, and a new Author Gravatar addon for author index pages. The 'Social Settings' metabox on the profile page allows users and admins to specify different descriptions for each social context (an alternative to the profile biography value), along with choosing a different image, either by entering an image ID or URL. The new Author Gravatar addon also pre-populates the Image URL field with the user's Gravatar image (if one exists).

Version 2.5.2 introduces a new 'Welcome' dashboard page, displayed once when the plugin or options are updated.

Version 2.5.3 adds a new 'Image Dimensions' option on the Pinterest publisher settings tab.

Version 2.5.4 (Upcoming) simplifies the settings pages by adding a new 'Plugin Settings to Display' option on the SSO Advanced settings page, and updates the admin-side CSS to improve the layout. The 'Plugin Settings to Display' value is set to 'Basic Plugin Options' by default, so if you find that some options have disappeared, you can change that option to 'All Plugin Options'.

Changelog

  • Enhancements
    • Added the "Read Me", "Setup Guide", and "What's New" setting pages to the Network Admin NGFB menu.

Version 2.5.3

  • Bugfixes
    • None
  • Enhancements
    • Added an 'Image Dimensions' option in the Pinterest publisher settings tab.
    • Added an additional check for images sizes returned by WordPress to validate the aspect ratio of uncropped images. WordPress has an unfortunate habit of returning images that are "close enough" and may return cropped images when an uncropped image was requested.

Version 2.5.2

  • Bugfixes
    • Fixed the post ID reference for header transient objects.
  • Enhancements
    • Renamed the 'About' settings page to 'Read Me'.
    • Added a new 'Setup Guide' settings page with configuration hints and suggestions.
    • Added a new 'Welcome' dashboard page, displayed only once, when the options are updated or the plugin is activated.
