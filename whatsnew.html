WPSSO version 2.5 includes two new features -- a 'Social Settings' metabox for the user profile page, and a new Author Gravatar addon for author index pages. The 'Social Settings' metabox on the profile page allows users and admins to specify different descriptions for each social context (an alternative to the profile biography value), along with choosing a different image, either by entering an image ID or URL. The new Author Gravatar addon also pre-populates the Image URL field with the user's Gravatar image (if one exists). The 'Social Settings' metabox can be removed by unchecking the 'Show Social Settings: User Profile' option on the Advanced settings page, and the Author Gravatar addon can be disabled by unchecking the 'Gravatar Images for Author Indexes' option on the General settings page.

Version 2.5.2 introduces a new 'Welcome' dashboard page, displayed only once, when the plugin or options are updated.
