WPSSO version 2.6.9.1 provides several new options.

A new 'Use Filtered (SEO) Titles' option allows the use of original title values in meta tags (by unchecking the option) instead of modified theme and/or SEO titles.

Two new 'Image URL Custom Field' and 'Video Embed HTML Custom Field' options have been added to the Advanced settings page, allowing you to fetch additional information from existing theme and/or 3rd-party plugin custom fields. These option custom field associations can be useful to reduce redundant entry fields on Post / Page editing pages.

A new 'Video Embed HTML' option has also been added to the Social Settings metabox, allowing you to paste complete video embed code instead of just a video URL.

Two new filters have also been added to allow finer-grain control over the addition / exclusion of the Social Settings metabox on specific post and/or use ID admin pages. A new constant is also available to manage the priority when saving post meta options (the default of 20 is after most other plugins have saved their post meta options).

WPSSO version 2.6.9.2 contains a small fix to escape special title separator values, and fixes the missing Google+ / Schema Description value on user profile pages.
